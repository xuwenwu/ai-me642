# Product Brief

## Purpose

AI-ME642 helps students practice responsible AI-assisted scientific computing. It makes modeling intent, AI assistance, simulation artifacts, validation evidence, and scientific interpretation visible to instructors.

## Phase I-Plus Goal

Build one reliable end-to-end workflow for **Lab 3: NVE Energy Conservation and Timestep Stability**.

## Primary Users

- Students: create project specs, disclose AI use, upload artifacts, run validation, explain results.
- Instructors/TAs: review evidence, grade with rubrics, export gradebook data.

## Success Criteria

- A seeded student can complete one Lab 3 submission from login to ZIP export.
- A seeded instructor or TA can review and grade that submission.
- Automated validation flags issues without pretending to be the final grade.
- Every AI-assisted artifact has a prompt/evidence path.
- Uploaded code is never executed by the server.

## Responsible AI Behaviors

Students must document:

- AI tool used.
- Prompt text.
- Output summary.
- Accepted parts.
- Rejected parts.
- Manual edits.
- Validation performed.
- Remaining concerns.

The system rewards evidence, reflection, and verification rather than unexamined AI output.
